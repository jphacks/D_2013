# README

## Setup
```
// おそらく不要
$ docker build -t jphacks2020 .

// ここから
$ yarn install

// Docker起動
$ docker-compose up

// Docker切る
$ docker-compose down

// 永続化きる場合
$ docker-compose down --volumes
```
