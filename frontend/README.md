# Frontend

React-Native / Expo で環境構築
基本ライブラリはyarnにてinstall

## setup
```
// expoをインストール
$ yarn global add expo
$ yarn global add expo-cli

// ライブラリをインストール
$ yarn

// 起動
$ cd src
$ yarn start

 -> QRコードが表示されるので、スマホで取って Expo client で表示させる
```

## server setup
```
// 知らんけど
$ cd server
$ yarn

// server起動する
$ node server.js
